# File Analyzer

A small C++ utility to perform simple analysis on files.

**Overview:**
- Purpose: quickly analyze file contents (lines, words, bytes) .
**Prerequisites:**
- A C++ compiler.

**Build (example with g++):**
```bash
g++ -o analyzer.exe "files pr.cpp"
```

**Run:**
```bash
./file_analyzer.exe <path-to-text-file>
```
